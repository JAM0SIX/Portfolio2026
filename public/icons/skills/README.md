# Skills / Stack icons

Drop icon files here. Each pill in `src/components/sections/SkillsStack/skillsData.js`
references a file in this folder by name via its `icon` field.

SVG is preferred (crisp, theme-neutral). PNG works too — just match the
filename. Icons render at 18×18; until a file exists the pill shows
label-only (the `<img>` hides itself on load error), so nothing breaks
if a file is missing.
